# DectctHideProcess
Linux下进程隐藏检测与定位工具

1.专注于Linux下隐藏进程的检测与实现手法的定位，并可以将隐藏的进程自动恢复，提升应急响应的效率,详细见：https://www.yuque.com/feiniao112/rq5bav/cs7r0cvps2eoua8u#XkDCu

2.可检测下面类型的进程隐藏：
  
  库文件劫持
  
  LKM Rootkit-Diamorphine
  
  进程挂载
